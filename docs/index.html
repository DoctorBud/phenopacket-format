A Vision

An Open and Computable Knowledge-sharing Technology designed for capturing and sharing structured bioinformatic knowledge. A solution to the Tower of Babel problem of diverse, sometimes siloed, biological databases, terminologies, and species.

A Technology

A language-neutral structured knowledge representation with encodings in JSON and YAML, and corresponding implementations and tools in several languages (Java, JavaScript, Python and More).

A Community

Supported by researchers, clinicians, model organism databases and commercial integration efforts.

The Vision of PhenoPackets

An Open and Computable Knowledge-sharing Technology designed for capturing and sharing structured bioinformatic knowledge.

A language-neutral data format specification and several language-specific implementations and tools.

The Technology of PhenoPackets

There are currently encodings of PhenoPackets specified for YAML and JSON, as well several implementations in various languages such as Java, JavaScript, and Python.

Source code and examples of PhenoPackets technology may be found in the following GitHub repositories:

  • PhenoPackets Organization
  • PhenoPackets Format
  • PhenoPackets Wiki
  • Java Reference Implementation
  • Python Tools
  • Java Tools

The Community of PhenoPackets

PhenoPackets were designed by, and intended to be used by, the diverse community of researchers, clinicians, educators and curators that rely upon rich, detailed, computable knowledge to do their work.
